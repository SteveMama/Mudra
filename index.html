    How to trigger detection

    1. Start the camera — green skeleton = hands tracked
    2. Make an ASL sign with clear hand movement
    3. Hold the sign still for ~0.5 s — orange border fires
    4. Result appears in the Translation panel

    Try these words (Word signs mode)

    Fingerspelling mode

    Switch the dropdown to Fingerspelling. Spell any word letter-by-letter using standard ASL handshapes (A–Z). The Squeezeformer model decodes the sequence when you hold still. Good for names and words not in the 2000-word vocabulary.
